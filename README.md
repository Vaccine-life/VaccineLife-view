# 슬기로운 백신생활

### Description

- '슬기로운 백신생활'은 백신 접종 후기를 자유롭게 공유할 수 있는 커뮤니티 사이트입니다.
- '백신 후기 게시판', '격리 후기 게시판', '의료진분들께' 게시판에서 코로나 및 백신과 관련된 다양한 경험을 자유롭게 공유할 수 있도록 하는 것에 중점을 두었습니다.
- 데스크탑, 모바일 웹사이트
